Building a Smart Attendance Logger with ESP32-CAM

Showcasing my project and what I learned

🔍 Overview

This project is an attendance logging system using the ESP32-CAM module and an FTDI programmer. It captures images of individuals and logs their entry with a timestamp, ideal for small-scale access control or attendance tracking.

🧰 Components Used

  • ESP32-CAM
  • FTDI USB-to-Serial Adapter
  • Breadboard and jumper wires
  • MicroSD Card
  • Optional: PIR sensor for motion detection

⚙️ How It Works

The ESP32-CAM takes a snapshot when triggered (either by a button or a sensor). Each image is saved to the SD card with a unique timestamped filename. Later, these records can be reviewed for attendance verification.
